Downloads

The current elbe version is 13.3

There are several ways to install a binary version of elbe.

The easiest way is to install the Debian packages via apt.

Debian Binary Packages

The recommended way to use ELBE is running it on a Debian PC. Linutronix provides a signed repository that can be included in apt.

Debian stretch

su -
cd /tmp
wget http://debian.linutronix.de/elbe/elbe-repo.pub
apt-key add elbe-repo.pub
echo "deb http://debian.linutronix.de/elbe stretch main" >> /etc/apt/sources.list
apt update
apt install elbe

Debian sid

su -
cd /tmp
wget http://debian.linutronix.de/elbe/elbe-repo.pub
apt-key add elbe-repo.pub
echo "deb http://debian.linutronix.de/elbe sid main" >> /etc/apt/sources.list
apt update
apt install elbe

Debian jessie

su -
cd /tmp
wget http://debian.linutronix.de/elbe/elbe-repo.pub
apt-key add elbe-repo.pub
echo "deb http://debian.linutronix.de/elbe jessie main" >> /etc/apt/sources.list
apt-get update
apt-get install elbe
